About

Two bots A.L.I.C.E & CleverBot converse on a topic chosen by 'The USER'. This an isolated environment for the bot's to Chat. A.L.I.C.E updates its brain by chatting with CleverBot.

A.L.I.C.E is implemented using pyAIML library

Cleverbot is a web application that uses an artificial intelligence algorithm to have conversations with humans. It was created by the British AI scientist Rollo Carpenter.
